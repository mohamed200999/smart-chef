# Smart Chef Uniform

هذا المشروع هو متجر يونيفورم الشيفات ويعمل الآن مع خادم محلي يشارك بيانات المنتجات بين الزوار.

## كيف تجعل الموقع حيًّا من أي جهاز

### 1. تشغيل الخادم المحلي

المشروع يحتوي على خادم Node.js جاهز في `server.js`.

إذا كان لديك Node.js مثبتًا:

```powershell
cd d:\smart
npm install
npm start
```

ثم افتح المتصفح على:

```text
http://localhost:3000
```

إذا أردت فتحه من الهاتف على نفس الشبكة، استخدم عنوان IP الخاص بجهاز الكمبيوتر بدلاً من `localhost`.

### 2. كيف يعمل الآن

- الزوار يشاهدون نفس المنتجات المضافة من لوحة الإدارة.
- المنتجات الجديدة والتعديلات تـُحفظ في `data/products.json`.
- يجب فتح الموقع عبر HTTP وليس عبر `file://`.

### 3. فتح لوحة الإدارة

اذهب إلى:

```text
http://localhost:3000/admin.html
```

- كلمة المرور الرئيسية: `2009`
- كلمة المرور للمدير (تسمح بتعديل المنتجات فقط): `5000`

### 4. هل تريد ربطه بـ Supabase؟

الآن الموقع يعمل بخادم محلي، ولكن إذا كنت تريد استخدام Supabase بدلاً من التخزين المحلي، يمكنني تعديل `js/supabase.js` لذلك.

### 5. استضافة الموقع على الإنترنت

إذا أردت أن يصبح الرابط متاحًا لأي شخص عبر الإنترنت، يمكنك استضافة المجلد على:

- Netlify
- Vercel
- أي استضافة تدعم صفحات ثابتة + Node.js

ويمكن أيضًا نشر الخادم على خدمة تدعم Node.js حتى يظهر الموقع كخدمة حية.

## ملاحظات

- تأكد من تشغيل الخادم أولاً باستخدام `npm start`.
- إذا لم يكن Node.js مثبتاً، ثبت Node.js من https://nodejs.org.
